# Rocketflix
Esse projeto foi retirado da plataforma rocketseat, da trilha discover, onde o desafio consiste em criar um indicador de filmes aleátorios, utilizando a API do <a href="www.themoviedb.org"> Themoviedb </a>

## Tecnologias
  - HTML
  - CSS
  - JavaScrip
